I chose to pursue a degree at UNL because of a few considerations.

  • Cost - UNL provides a great value
  • Software Engineering program - many places just offer a CS degree, but UNL is unique in this regard.

Relevant coursework:

  • Software Engineering core classes - a series of courses teaching the basics of programming up to advanced system architecture. Topics such as algorithms and discrete math were taught along the way
  • Computer Science tech electives - Classes such as processor micro-architecture and programming language design helped round out understanding of the tools we use each day for Software Engineering
